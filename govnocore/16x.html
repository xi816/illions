Registers

This section shows all Govno Core 16X registers.

Program counter

The program counter is a 16-bit register that stores the current instruction being executed.

It can be modified using jumps, calls and returns

Instructions

Instruction size

Instruction size can be 1 bit (like RET - $33) or 2 bit (like LDA - $6605). 2-byte instructions in Govno Core 16X are called "word instructions". First byte of a word instruction defines its type (ex. LDA starts with $66, and page $66 is a register-manipulating operation).

Page Meaning
$0F Control flow operation
$10 Stack operation
$66 Register operation

ADD - Add values

The instruction adds the contents of two registers/memory location/immediate.

Addressing mode Opcode Instruction Size

AND - Bitwise AND

Performing the bitwise AND to two registers.

Addressing mode Opcode Instruction Size
rc8 $10D8 Bitwise and for %rc0 and %rc1 3

CLC - Clear carry

Clearing the carry flag.

Addressing mode Opcode Instruction Size
Implied $36 Clear carry 1

CMP - Compare

Compare registers/immediate values and set the flags.

Flags after the instruction:

C Carry flag Not affected
Z Zero flag Set if values are equal

Addressing mode Opcode Instruction Size
rc8 $10F6 Compare %rc0 and %rc1 3
reg8,imm16 $10EE Compare %r and $i 3

LDA - Load into A register

The instruction loads the contents of register/memory location/immediate into the A register.

Addressing mode Opcode Instruction Size
imm16 $6605 Set %a to $i 4
reg8 $6641 Set %a to %r 3
addr16 $6692 Set %a to [$i] 4
